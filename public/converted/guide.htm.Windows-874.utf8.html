คําแนะนําในการใช้โฮมเพจ"ปฏิจจสมุปบาท"

เครื่องหมายที่ควรรู้      = คลิกที่เม้าส์ด้านขวาเพื่อ Link ได้, 

คลิกที่นี่ เพื่อเพิ่มหน้านี้ในแฟ้ม Favorites. เพื่อความสะดวกในการเปิดครั้งต่อไป

คลิกที่นี่ เพื่อตั้ง ปฏิจจสมุปบาท เป็นหน้าแรกที่เข้า Internet(Home Page)

คลิกที่นี่ สําหรับ Webmaster ที่ต้องการ Banner ของ Web นี้

คลิกที่นี่ เพื่อแจ้งข้อบกพร่อง ข้อผิดพลาด เพื่อการแก้ไข

Pop-up Controls ปิดชั่วคราว หรือกด Ctrl + คลิกที่ต้องการเปิด เมื่อจะเปิดหน้าต่าง(Windows)ย่อย

ปรับ"ขนาดภาพ" อัติโนมัติ  ตามความละเอียดของจอภาพที่ใช้   ถ้าเปิดเข้ามาทางหน้าแรกของเว็บ  

(หรือใช้วิธีกด Ctrl + หมุนเม้าส์ เป็นวิธีขยายตัวอักษรให้ ใหญ่ หรือ เล็ก ได้ตามต้องการ)

การ Copy ให้ลากคลุมบริเวณที่ต้องการ  แล้วคลิกที่แถบเครื่องมือมุมบนซ้าย Edit ---> Copy

ขนาดตัวอักษร-Text Size = Medium (อยู่ใน View)  บางบทสามารถปรับขนาดอักษรได้ที่

Back ซ้ายสุดในแถบคําสั่งข้างบน (Tool bar) ใช้ในการสั่ง "กลับไปที่หน้าเดิม" ของทั้งโฮมเพจนี้
